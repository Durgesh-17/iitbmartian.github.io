Control Systems for rover

Written primarily for closed loop operation of Arm and Steer

  • All 6 wheels are driven independently from GPIO pins of Raspberry Pi 3 in open loop, and rear 4 wheels are steered using closed loop control (see fig. on left)
  • Steering is controlled by Roboclaw Motor Drivers onto which we employ a Type II control system, to track ramp input (see fig. on left, tracking of i/p, dotted line, is done by o/p, solid purple)
  • We are working towards getting a state space description of the same, so that we can employ Inverse Kinematics or LQR control for the arm, using Moveit! motion planning plugin (see fig. on left)
